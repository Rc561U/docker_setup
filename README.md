A simple PHP setup with Nginx and MySQL Using Docker and Docker Compose 

- PHP - version 8.1

- Nginx - 1.23.1

- Mysql - version 8.0.30 
<hr>

Docker compose command to start:
```
sudo docker compose up -d
```

to stop:
```
sudo docker compose down
```
to get logs from php container
```
sudo docker compose logs php    
```
to get information about running services
```
sudo docker compose ps
```
